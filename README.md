##Feature

- 导入QQ音乐
- 丰富多彩的样式
- 颜色切换
- 一键复制、清空
- 离线使用

##Technical Stack

- Ractive (template-driven UI library)
- Require.JS (AMD)
- CKEditor
- jQuery Nicescroll
- jQuery Mixitup
- Spectrum (ColorPicker)

###Test Stack

- Mocha
- RequireJS

Browser: Chrome / Firefox / IE8 +
